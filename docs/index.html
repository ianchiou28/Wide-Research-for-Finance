项目概述

Wide Research for Finance 是一个基于人工智能的自动化财经情报系统，旨在帮助投资者、分析师和金融从业者高效获取和分析全球财经资讯。

核心价值

  • 信息聚合：自动从17个主流财经媒体和官方机构采集新闻
  • 智能分析：使用DeepSeek AI进行情感分析、实体识别和事件分类
  • 实时监控：每小时自动运行，第一时间捕捉市场动态
  • 成本优化：相比Bloomberg Terminal节省99.5%成本

核心功能

全球数据采集

支持17个RSS源和4个官方网站的自动爬取，覆盖国际财经、专业分析、科技加密等多个领域。

RSS订阅 网页爬虫

AI智能分析

基于DeepSeek大语言模型，自动进行情感分析、实体提取、事件分类和影响力评估。

情感识别 实体提取 事件分类

知识整合

计算情绪指数、统计热点实体、评估事件影响力，生成结构化的市场洞察。

情绪指数 热点统计

自动化报告

每小时自动生成简报，支持邮件推送或本地存储，确保信息及时触达。

定时任务 邮件推送

核心模块说明

模块 功能 关键技术
collector.py 从17个RSS源采集新闻数据 feedparser, yaml
web_scraper.py 爬取美联储、SEC等官方网站 BeautifulSoup, requests
processor.py AI批量分析新闻内容 DeepSeek API, 批量处理
report_generator.py 生成结构化报告 情绪统计, 热点提取
email_sender.py 邮件推送或本地保存 smtplib, Gmail SMTP
main.py 主程序和任务调度 schedule, 多模式运行

安装部署

环境要求

  • Python 3.8 或更高版本
  • DeepSeek API密钥（从 platform.deepseek.com 获取）
  • 可选：Gmail账户（用于邮件推送）

安装步骤

  1. 克隆仓库
    git clone https://github.com/ianchiou28/Wide-Research-for-Finance.git cd Wide-Research-for-Finance
  2. 安装依赖
    pip install -r requirements.txt
  3. 配置环境变量
    cp .env.example .env # 编辑 .env 文件，填入以下配置： DEEPSEEK_API_KEY=sk-your-key-here EMAIL_FROM=your_email@gmail.com # 可选 EMAIL_PASSWORD=your_app_password # 可选 EMAIL_TO=recipient@email.com # 可选
  4. 运行程序
    python main.py
