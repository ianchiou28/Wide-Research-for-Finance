## 🔧 维护指南

### 修改颜色
编辑 `static/css/common.css` 中的 CSS 变量:
```css
:root {
    --primary: #667eea;
    --secondary: #764ba2;
    /* ... */
}
```

### 添加新组件
1. 在对应的 HTML 文件中添加结构
2. 在 `<style>` 标签中定义样式
3. 在 `<script>` 中添加交互逻辑

### 调试技巧
- 打开浏览器开发者工具
- 查看 Console 面板的日志输出
- 使用 Network 面板检查API调用
- Elements 面板实时调整样式
